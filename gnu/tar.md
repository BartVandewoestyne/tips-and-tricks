# GNU tar

## Creating archives

Create a gzip-compressed archive (`.tar.gz` or `.tgz`):

```text
tar -czvf myarchive.tar.gz folder/
```

Create a bzip2-compressed archive (`.tar.bz2`):

```text
tar -cjvf myarchive.tar.bz2 folder/
```

Create an xz-compressed archive (`.tar.xz`):

```text
tar -cJvf myarchive.tar.xz folder/
```
